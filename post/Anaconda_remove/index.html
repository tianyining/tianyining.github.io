Ubuntu20.04卸载并重新安装Anaconda

说明

台式机 Ubuntu18.04，Anaconda 无法使用 conda 命令更新或安装，尝试多次未找出原因，因此重装 Anaconda 。

卸载 Anaconda

卸载 anaconda 的步骤如下：

  • 删除整个 anaconda 的目录；
1

$ rm -rf ~/anaconda3

  • 清理 ~/.zshrc 文件中之前添加的和 conda 有关的所有路径；

  • 使其生效，在终端执行：

1

$ source ~/.zshrc
  • 关闭终端，然后再重启一个新的终端，这一步很重要，不然在原终端还会显示 anaconda 激活现象；

重新安装 Anaconda

直接运行 bash 脚本安装，询问是否 conda init 本次安装直接空格跳过，即没有将命令行工具 conda 添加到系统的 PATH 环境变量中。那我们可以手动添加😊：

1
2

$ conda init zsh
$ code ~/.zshrc   			## 查看 ~/.zshrc 的变化

遗留问题

重新安装 Anaconda 后，在 base 环境下仍然无法更新、安装，需要重新创建一个虚拟环境，比如 PKU_Summer 才可安装、升级包。
