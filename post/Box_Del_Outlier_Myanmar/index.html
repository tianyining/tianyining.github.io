说明

在频散曲线质量控制时，最初采用计算不同整数周期相速度平均值以及标准差的方法，将偏离 2~3 倍标准差以外的数据点剔除。但是这种判别处理的原理及方法仅限于对正态或近似正态分布的样本数据进行处理，以及当数据量较大时才适合使用。

现实情况是，频散数据不一定符合正态分布，如果用几倍的 σ 去检测异常值(outlier)就不合适，那么我们如何来判断这些变化是否在合理的范围呢？可以考虑使用箱型图，箱型图的四分位距（IQR）对异常值进行检测，也叫Tukey’s Test。我们可以看下如下箱型图的样式：

箱型图中间是一个箱体，也就是粉红色部分，箱体左边，中间，右边分别有一条线，左边是下四分位数（Q1），右边是上四分位数（Q3），中间是中位数（Median），上下四分位数之差是四分位距（称IQR）。IQR也可以说指的是两个四分值之间的范围大小。用 Q1-1.5IQR 得到下边缘（最小值），Q3+1.5IQR 得到上边缘（最大值），在上边缘之外的数据就是极大异常值，在下边缘之外的数据极小异常值，总之在上下边缘之外的数据就是异常值。另外要说明的是图中的数值 1.5 是一个可变的系数，表示的是中度异常；对于重度异常的情况，可以视情况而定将系数提高至 3。

构建箱型图识别数据中的异常值，并对异常值进行剔除 & 可视化。

局限性

使用箱型图识别异常值也存在一定的局限性，比如不能精确地衡量数据分布的偏态和尾重程度；对于批量比较大的数据，反映的信息更加模糊以及用中位数代表总体评价水平有一定的局限性。

本质：用中位数代表总体评价水平，有一定的局限性；

        'color': 'blue',
    }) # boxprops

plt.xlim(0, 7)
plt.xlabel("PhaseV(km/s)", size=24)
ax.set_title('PhaseV Box Plot Period 26s', fontsize=26)
plt.savefig(
    "../Figs/PhaseV Box Plot Period 26s.png",
    dpi=720,
)

import os
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from scipy import stats
from jupyterthemes import jtplot

jtplot.style(theme="monokai")

plt.rcParams['font.sans-serif'] = ["Times New Roman"]
plt.rcParams['axes.unicode_minus'] = False
plt.rcParams["axes.linewidth"] = 1  # 图框宽度
plt.rcParams["figure.dpi"] = 720  # plt.show() 图片分辨率
'''
 读入 26s Myanmar 周期的相速度数据；
'''
input_disp_dir = "../DISP_Period_Myanmar/"

namescol = ["GroupV", "PhaseV", "Sta_Pair"]
dataset = pd.read_table(input_disp_dir + "26",
                        delim_whitespace=True,
                        names=namescol)
PhaseV = dataset.PhaseV
PhaseV = pd.DataFrame(PhaseV)
PhaseV.head()
'''
 构建散点图，并通过 describe() 函数获得数据的统计信息量；
'''
data_des = PhaseV["PhaseV"].describe()
q1 = data_des["25%"]
q3 = data_des["75%"]
iqr = q3 - q1
minimum = q1 - 1.5 * iqr
maximum = q3 + 1.5 * iqr
outlier = PhaseV[(PhaseV["PhaseV"] < minimum) | (PhaseV["PhaseV"] > maximum)]

correct = PhaseV[(PhaseV["PhaseV"] >= minimum) & (PhaseV["PhaseV"] <= maximum)]

print("26s Period PhaseV 共有 {} 个异常值".format(len(outlier)))
print("26s Period PhaseV 共有 {} 个正常值".format(len(correct)))
'''
 Scatter Plot Begin !!! 
'''
fig = plt.figure(figsize=(12, 4))

# First, correct scatter plot
plt.scatter(
    correct.index,
    correct.values,
    s=50,
    color="green",
    edgecolors="black",
    alpha=0.5,
    label="normal",
)

# Secondly, outlier scatter plot
plt.scatter(
    outlier.index,
    outlier.values,
    s=50,
    color="red",
    edgecolors="black",
    alpha=0.5,
    label="outlier",
)
'''
 添加 x 轴上下限横轴标线；
'''
plt.axhline(y=maximum, color="blue", linestyle="--", label="upper")
plt.axhline(y=minimum, color="blue", linestyle="--", label="lower")
plt.legend(loc=8, ncol=4)
plt.title("26s Period PhaseV Scatter Plot", size=26)

# 设置坐标刻度值的大小
plt.tick_params(labelsize=20)
plt.xlabel("index", size=20)
plt.ylabel("PhaseV(km/s)", size=20)
plt.grid(linestyle="--", alpha=0.3, color="grey")
plt.savefig("../Figs/26s Period PhaseV Scatter Plot_Box.png")

import os
import glob
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
'''
 读入 ALL Period Myanmar Disp Files；
'''
input_disp_dir = "../DISP_Period_Myanmar/"
output_disp_dir = "../Del_outlier_Myanmar_DISP/"

for sgl_disp in glob.glob(input_disp_dir + "*"):
    sgl_disp_name = os.path.basename(sgl_disp)
    namescol = ["GroupV", "PhaseV", "Sta_Pair"]
    dataset = pd.read_table(input_disp_dir + sgl_disp_name,
                            delim_whitespace=True,
                            names=namescol)
    PhaseV = dataset.PhaseV
    PhaseV = pd.DataFrame(PhaseV)
    PhaseV.head()

    data_des = PhaseV["PhaseV"].describe()
    q1 = data_des["25%"]
    q3 = data_des["75%"]
    iqr = q3 - q1
    minimum = q1 - 1.5 * iqr
    maximum = q3 + 1.5 * iqr

    dataset_new = dataset[(dataset["PhaseV"] >= minimum)
                          & (dataset["PhaseV"] <= maximum)]
    dataset_new.to_csv(
        output_disp_dir + sgl_disp_name,
        sep='\t',
        index=None,
        header=None,
    )

import os
import glob
import numpy as np
import matplotlib.pyplot as plt
from jupyterthemes import jtplot

jtplot.style(theme="monokai")

plt.rcParams["axes.linewidth"] = 1  # 图框宽度；
plt.rcParams["figure.dpi"] = 720  #
plt.rcParams['font.sans-serif'] = ['Times New Roman']

input_dir = "../Del_outlier_Myanmar_DISP/"
output_dir = "../Figs/"

x = []
y = []
for sgl_disp in glob.glob(input_dir + "*"):
    sgl_disp_name = os.path.basename(sgl_disp)
    x.append(float(sgl_disp_name))

    tmpinfos = np.loadtxt(sgl_disp, delimiter="\t", usecols=(0, 1))

    GroupV = tmpinfos[:, 0]
    PhaseV = tmpinfos[:, 1]
    PhaseV_tuple = tuple(PhaseV)
    y.append(PhaseV_tuple)

'''
 Plot Begin !!!
'''
fig = plt.figure(figsize=(12, 8))
for xe, ye in zip(x, y):
    plt.scatter(
        [xe] * len(ye),
        ye,
        s=8,
        color="green",
        edgecolors="black",
    )
plt.grid()
plt.xlabel("Period(s)", size=25) 
plt.ylabel("PhaseV(km/s)", size=25)
plt.tick_params(labelsize=15)  # 设置坐标刻度值的大小
plt.title("ALL Period PhaseV After Del Outlier", size=25)
plt.savefig("../Figs/ALL Period PhaseV Scatter Plot After Del Outlier.png")
