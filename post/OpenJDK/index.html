Ubuntu18.04安装OpenJDK报错

  • 参考链接

说明

Ubuntu系统下运行 Java 程序需要安装 Java 运行环境，即 OpenJDK：

1

$ sudo apt install default-jdk

报错显示：

1

$ E:Unable to locate package default-jdk

解决方法

I used another repository for oracle java, and it works.

1
2
3

$ sudo add-apt-repository ppa:linuxuprising/java
$ sudo apt-get update
$ sudo apt install  oracle-java11-installer
