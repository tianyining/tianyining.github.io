下载地址：https://www.sublimetext.com/3

安装 Package Control

为什么要安装 Package Control ?

Package Control是一个管理Sublime Text插件的插件，通过Package Control我们可以直接使用Sublime Text控制台来安装其他插件，快捷简单。

Package Control 安装方法

理论上有两种方式可以安装 Package Control ，一种方式是打开控制台，复制粘贴代码回车即可，但是可能是由于网络连接不畅，这种方法 fail，本次我们选择从源码安装。

1. Package Control 源文件下载

下载地址：https://github.com/wbond/package_control

点击右上角的Clone or download按钮，然后选择Download ZIP

2. 将下载的ZIP文件解压，并重命名为Package Control，未重命名或重命名错误则会出现如下错误提示。

3. 在菜单栏中点击Preferences > Browe Packages...，打开一个文件夹，复制刚才的Package Control文件至此目录下。

4. 在菜单中点击Preferences，若出现Package Settings和package control选项，就说明安装Package Control成功了。

Sublime Text 3 插件安装

如何安装插件？

通过快捷键Command + Shift + P打开 Package Control 来安装插件。在打开的输入框中输入install，会根据输入状态自动提示，用鼠标点击或键盘回车选择 Install Package 进入查找安装界面：

输入你想要安装的插件名称：

Sublime Text 3 运行环境配置

目标：决定利用Sublime Text 3 + Anaconda插件搭建一个Python的IDE。

  2. 在Sublime新建build system，编辑.sublime-build文件 ：
