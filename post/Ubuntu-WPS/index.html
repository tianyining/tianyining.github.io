Ubuntu安装WPS

2021-01-31 23:38
Ubuntu
约 330 字 预计阅读 1 分钟 次阅读

文章目录

  • 参考链接

Ubuntu 自带的 LibreOffice 具有简单的文档查看和编辑功能，但其兼容性一般。兼容性更好的是 WPS Office。

  1. 下载64位deb格式的安装包：WPS Office for Linux 官网

  2. 安装WPS：

    1

    sudo dpkg -i wps的deb包

    在打开wps writer的时候提示有些公式无法正常显示，因为缺少symbol等字体。

  3. 解决字体问题

    下载缺失的字体文件，然后复制到系统中的/usr/share/fonts文件夹中。(下载地址)

    在~/Downloads目录下新建一个文件夹font，并将下载的压缩包解压到这个文件夹中。

    1
2

    mkdir font
unzip -d font wps_symbol_fonts.zip

    将这个目录下所有字体都复制到/usr/share/fonts

    1

    sudo cp font/* /usr/share/fonts

    再执行以下命令：

    1
2
3
4

    sudo mkfontscale
sudo mkfontdir               // 生成字体的索引信息
   
sudo fc-cache              // 更新字体缓存

    接下来重启wps，就不会再出现字体缺失的提示了。
