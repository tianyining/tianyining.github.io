Ubuntu 安装 VScode

官方安装说明

为 VScode 安装插件

VScode 的安装插件可以在 Extensions 中搜索然后进行安装。

推荐安装的插件：

  • Code Runner ：支持代码一键运行；⭐️

为支持中文脚本，Code Runner中需要设置终端输出：

在Extensions菜单中，点击Code Runner插件的齿轮图标，弹出菜单中选择“Extension Settings”，勾选“Run in Terminal”。另外勾选“ Save File Before Run”，可在执行脚本之前先自动保存。

  • Python ：Python 解释器；⭐️（Jupyter ）
  • Code Spell Checker
  • Document This
  • One dark Pro
  • Pylint
  • Pylance
  • Vscode-icons
  • Vscode-pdf
  • ESLint
  • Beautify
  • Debugger for Chrome

安装完插件后，需要重新启动 VScode。
