什么是 Jupyter Notebook?

详细内容请参考：geting started with conda

Jupyter Notebook 之前被叫做 IPython Notebook，是基于 Python 环境下的一套集成了代码执行，富文本编辑，数学符号，以及多媒体影音嵌入的交互环境。这个项目广泛受到大家的喜爱，现在已经从单纯的 Python 基础发展成为更广泛的 Jupyter Lab ，可以使用更多的语言在 Jupyter 中运行，并且获得实时的结果。

管理虚拟环境

我们可以用 anaconda 来创建独立的 Python 环境。

1
2
3
4

$ conda init
## 重新开启 terminal
$ conda list 				# 可以看到当前 Python 环境下已安装的 module 
$ conda env list		# 查看所有的环境

创建自己的虚拟环境

我们当然不满足一个 base 环境，需要建立一个独立的 python 编译环境，而不是使用 base，这是为了避免各种 package 版本之间相互污染，建议针对不同的项目每一次都进行不同的环境进行区分。

创建一个名称为 PKU_Summer 的虚拟环境，并指定 python 版本为 python3（这里 conda 会自动找 Python3 中最新的版本下载）。

1

$ conda create --name PKU_Summer python=3

于是我们就拥有了一个 PKU_Summer 的虚拟环境，现在的这个虚拟环境除了 python 自带的一些官方包之外是没有其他包的，是一个比较干净的环境，接下来我们需要切换到这个环境。

1
2

$ conda env list
$ conda activate PKU_Summer

然后你会看到：

1

> (PKU_Summer) tianyining@tianyining-ThikCentre-M8600t-D065

接下来安装在这个虚拟环境中我们需要用到的包：

1
2
3
4
5

$ conda install -c conda-forge jupyterlab
$ conda install numpy
$ conda install -c conda-forge scipy
$ conda install -c conda-forge matplotlib
$ conda install -c conda-forge obspy

安装其他任何python package :

1

$ conda install *****
