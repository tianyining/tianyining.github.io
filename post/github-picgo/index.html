sudo ln -s /usr/local/lib/nodejs/bin/node /usr/bin/node

否则，加上sudo时，系统会找不到node与npm。

安装 PicGo-GUI

去Molunerfinn/PicGo/releases下载最新版本的PicGo，我此时下载的最新版本为PicGo 2.3.0-beta.3。

此处以我的版本为例，说明在ubuntu系统下安装使用PicGo。

  • 点击下载PicGo-2.3.0-beta.3.AppImage
  • 右键PicGo-2.3.0-beta.3.AppImage属性，选中Allow executing file as program.、
  • 右键PicGo-2.3.0-beta.3.AppImage，点击run，默认下面图标，不要着急，不要一气之下把它给删了，继续…
  • 鼠标放在上面的小图标上，右键，打开详细窗口：

参照PicGo+Gitee搭建个人图床，新建放置图床的仓库、获取私人令牌token

  • 配置完，可回到上传区，将图片拖到虚框内，如果提示上传成功，可去个人 GitHub 对应仓库下查看刚上传的图片是否成功。

在 Typora 中插入已经倒入GitHub仓库中的图片资源

  1. 将 markdown 需要用的图片放置在 git 仓库中，发布到 GitHub 上；
  2. 访问 GitHub 仓库：https://github.com/tianyining/picgo/tree/master/img
  3. GitHub中访问想要插入的图片，点 Download 后，选择copy image address;
  4. 在 Typora 中，输入如下语法，即可成功插入图片。
