说明

这篇 blog 记录 jupyter notebook 下的使用技巧。

1. jupyter notebook 深色主题背景下图表坐标看不清

将 jupyter notebook 设置为深色背景，但深色背景画图就会遇到图表坐标文字显示看不清的问题。

解决方案

代码开头加上两行，换一个主题即可

1
2
3
4
5

$ from jupyterthemes import jtplot
$ jtplot.style(theme = 'monokai') ## 选择一个绘图主题即可

## 除了 jtplot 以外，还可以直接使用
$ plt.style.use('seaborn-darkgrid')  ## 设置画图的风格

参考链接

  • https://blog.csdn.net/haley_yuen/article/details/108429708

2. 解决中文显示&负号显示问题_MacOS

plt.rcParams['axes.unicode_minus']=False    # 用来正常显示负号

3. Jupyter Notebook 设置黑色背景主题、字体大小、代码自动补全

背景主题、字体大小设置

1

$ pip install jupyterthemes

更新 jupyter 主题

1

$ pip install --upgrade jupyterthemes

查看可用主题：

1

$ jt -l

下面是我常用的背景主题，选择 monokai ，支持语法高亮：

1

$ jt -t monokai -f fira -fs 15 -cellw 60% -ofs 11 -dfs 11 -T -N

-f(字体) -fs(字体大小) -cellw(占屏比或宽度) -ofs(输出段的字号) -T(显示工具栏) -N(显示自己主机名)

4. Markdown 如何在 jupyter notebook 里面改变字体、大小、颜色以及高亮

改变字体颜色(color)

红色+加粗

1

**<font color = red>红色+加粗</font>**

蓝色+斜体

1
2
3

*<font color = blue>蓝色+斜体</font>*
或
_<font color = blue>蓝色+斜体</font>_

绿色+加粗+斜体

1
2
3
4
5

***<font color=green>绿色+加粗+斜体</font>***
或
_**<font color=green>绿色+加粗+斜体</font>**_
或
___<font color=green>绿色+加粗+斜体</font>___

改变字体(face)、大小(size)

5号雅黑

1

<font color=steel size=5 face=雅黑>5号雅黑</font>
  • 可以看出来 color, size, font 的顺序可以随意调整，且参数不需要加引号

参考链接

  • https://blog.csdn.net/AlanGuoo/article/details/86318381

4. 解决 matplotlib 中文显示问题 & 安装英文字体_Ubuntu18.04

安装 SimHei 字体(可支持中文显示)

在 jupyter notebook 中如果想使用 SimHei 字体，可能需要自行安装，否则会报错：

1

Font family [‘sans-serif‘] not found.Falling back to DejaVu Sans.

安装步骤如下：

1. 在conda 环境终端下jupyter notebook中也可 ，查看字体路径

1
2

import matplotlib    
print(matplotlib.matplotlib_fname())

类似输出如下：

1

/opt/anaconda3/envs/PKU_Summer/lib/python3.8/site-packages/matplotlib/mpl-data/matplotlibrc

2. 下载 SimHei 字体

官网下载地址

将下载好的 SimHei ttf 移动到指定目录下，指定目录一般为：

1

/opt/anaconda3/envs/PKU_Summer/lib/python3.8/site-packages/matplotlib/mpl-data/fonts/ttf

3. 删除 matplotlib 的缓冲目录

首先查看 matplotlib 的字体缓存路径：

1
2

import matplotlib 
matplotlib.get_cachedir()

删除缓存路径：

1

rm ~/..cache/matplotlib -fr

4. 修改 matplotlibrc 文件

打开 matplotlibrc 文件：

文件路径：

1

/opt/anaconda3/envs/PKU_Summer/lib/python3.8/site-packages/matplotlib/mpl-data/matplotlibrc

修改如下内容：

1
2
3
4
5
6

 font.family         : sans-serif   
 # 去掉前面的#     
 font.sans-serif     : SimHei, Bitstream Vera Sans, Lucida Grande, Verdana, Geneva, Lucid, Arial, Helvetica, Avant Garde, sans-serif  
 # 去掉前面的#，并在冒号后面添加SimHei
 axes.unicode_minus  : False
 # 去掉前面的#，并将True改为False

5. 重新启动 jupyter notebook
