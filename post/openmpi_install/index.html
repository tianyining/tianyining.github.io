Openmpi在Ubuntu20.04上的安装

下载 OpenMPI

在官网上下载最新版本的安装包.

解压并进行配置

1
2
3
4
5

$ tar -zxvf openmpi-X.X.X.tar.gz

$ cd openmpi-X.X.X

$ ./configure --prefix="/usr/local/openmpi"

注意最后一行是将其安装到 /usr/local/openmpi目录下，可以指定为其他目录。

Build 并安装

1
2
3

$ make -j8         ## 表示用 8 核进行编译

$ sudo make install

如果 make 的时候不加 -j8 会需要很久。

添加环境变量

在 ~/.zshrc 中添加下列语句：

1
2

export PATH="$PATH:/usr/local/openmpi/bin"
export LD_LIBRARY_PATH="$LD_LIBRARY_PATH:/usr/local/openmpi/lib/"

保存后，执行：

1
2

$ source ~/.zshrc
$ sudo ldconfig

打开新的终端，使环境变量生效。

测试是否安装成功

测试方式一：

1

3
4

$ mpirun

测试方式二：

第一种测试方式也许并不直观，我们可以下个 mpi 程序测试下安装是否成功了。

test.cpp

 1
 2

 5

 9
10

15

#include <mpi.h>
#include <stdio.h>
int main(int argc, char**argv){
    //openMPI的初始化函数
    MPI_Init(&argc, &argv);
    int world_size, wrank;
    //获取容器中进程数
    MPI_Comm_size(MPI_COMM_WORLD, &world_size);
    //获取当前进程标识
    MPI_Comm_rank(MPI_COMM_WORLD, &wrank);
    printf("Hello world. I'm the process %d, we are %d process in total\n", wrank, world_size);
    //openMPI的结束函数
    MPI_Finalize();
    return 0;
}

然后运行：

1
2

$ mpicxx test.cpp -o test
$ mpirun -n 4 ./test

mpicxx 是编译 c++ 代码，编译 c 代码是用 mpicc

mpirun 是运行 mpi 程序，-n 4 表示创建 4 个进程。
