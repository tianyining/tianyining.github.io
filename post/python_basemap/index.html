Basemap 环境安装(初稿)

说明

在 python=3.9 的 pygmt 环境中无法安装 basemap ，因此创建新的 conda 环境。

1
2
3

$ conda create --name basemap 
$ conda activate basemap 
$ conda install -c anaconda basemap

创建环境

跑程序包的时候，首先需要创建环境，run 这些脚本时需要注意的是需要安装 basemap 以及 matplotlib==3.1.0 or lower。

1

This code requires the basemap package and requires Matplotlib 3.1 (or below)

另外需要注意的是，obspy 和 geopy 库安装成功后，分别需要再次安装 matplotlib==3.1.0，因为 obspy 和 geopy 安装的时候会更新 matplotlib 的版本，但是该脚本需要用到低版本的 matplotlib。

$ conda install scipy 
$ conda install obspy 
$ conda install -c conda-forge basemap-data-hires (run fig1.py 时提示)
$ pip install matplotlib==3.1.0 
$ conda install -c conda-forge geopy 
$ pip install matplotlib==3.1.0
