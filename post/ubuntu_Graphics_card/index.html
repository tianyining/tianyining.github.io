写在最前面

在Ubuntu终端手贱安装了Nvidia显卡驱动，重新开机后发现无法进入图形可视化界面，通过以下操作拯救了Ubuntu系统。

解决步骤

  • 开机后，疯狂摁shift键，进入像是windows bios界面一样的grub界面：

  • 选择 Ubuntu advanced，enter进入后选第二行 recovery mode 按 e 编辑。将nouveau的参数修改，在GRUB_CMDLINE_LINUX_DEFAULT那一行加上如下语句：

    1

    nouveau.modeset=0

    按照下面提示保存修改，并重启系统，ctrl+x（遗忘，不准确）

    然后进入了恢复模式，修复了dpkg

  • 进入tty模式（莫名其妙能进入了tty模式）

    输入账户名：tianyining

    Password:

  • 运行以下语句

1
2
3

sudo apt-get purge nvidia-*
sudo apt-get update
reboot

重新启动后，即可进入图形界面。

如何进入tty模式

进入tty终端模式：ctrl+alt+f2

返回到图形界面：ctrl+alt+f1

使用ctrl+alt+f3~f6可以打开tty控制台
