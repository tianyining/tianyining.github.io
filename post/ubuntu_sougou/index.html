Ubuntu18.04搜狗拼音安装

Ubuntu

    • 安装fcitx

安装步骤简述

  • 安装fcitx

1
2
3

sudo apt-get install fcitx-bin

sudo apt-get install fcitx-table

  • 下载搜狗拼音输入法 for linux

    在搜狗输入法的网站下载安装包：https://pinyin.sogou.com/linux/

  • 安装

    打开.deb安装包的下载目录，执行如下安装命令

1

sudo dpkg -i sogoupinyin_2.2.0.0108_amd64.deb

  • 解决安装依赖问题

    如果不存在依赖问题，此步骤可以跳过

    在终端执行如下命令：

1

sudo apt-get install -f

注意：msi 笔记本双系统中的 Ubuntu18.04 在实际安装过程中只有安装 sogoupinyin_2.2.0.0108_amd64.deb 这个版本时才成功，因此需对应版本安装。（sogoupinyin_2.2.0.0108_amd64.deb 已经存入 2T），需要注意的是，该版本的 sogoupinyin 与 pycharm 不兼容

但是在台式工作机上 Ubuntu18.04 成功安装最新版本的 sogoupinyin_2.4.0.3469.deb，成功解决 sogoupinyin 与 pycharm 安装时不兼容的问题。

  • 设置搜狗输入法

第一步：设置——>区域和语言——>管理已安装的语言，将键盘输入法系统设置为fcitx，然后点击上面的Apply System-Wide应用到全局，重新启动。

第二步：重新启动完毕后，点击右上角的小键盘，设置当前输入法（Configure Current Input Method）——>添加输入法——>添加搜狗拼音输入法，并置顶。

进入到Add input method界面，将下面的Only Show Current Language 点掉后，在搜索栏搜索搜狗拼音，选中之后进行添加。

添加成功后，将搜狗拼音移到第一位。
